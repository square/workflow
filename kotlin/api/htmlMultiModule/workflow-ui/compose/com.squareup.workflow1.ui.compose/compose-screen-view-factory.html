@WorkflowUiExperimentalApi
inline fun <RenderingT : Screen> composeScreenViewFactory(noinline content: @Composable (rendering: RenderingT, environment: ViewEnvironment) -> Unit): ScreenViewFactory<RenderingT>

Creates a ScreenViewFactory that uses a Composable function to display the rendering.

Simple usage:

val FooViewFactory = composeScreenViewFactory { rendering, _ ->
  Text(rendering.message)
}
