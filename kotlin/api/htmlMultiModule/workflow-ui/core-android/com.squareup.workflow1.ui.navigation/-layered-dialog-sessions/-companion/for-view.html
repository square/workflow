fun forView(view: View, superDispatchTouchEvent: (MotionEvent) -> Unit): LayeredDialogSessions

Creates a LayeredDialogSessions instance based on the given view, which will serve as the source for a LifecycleOwner, and whose bounds will be reported via OverlayArea.

  • The view's dispatchTouchEvent and dispatchKeyEvent methods should be overridden to honor LayeredDialogSessions.allowBodyEvents.

  • The view's onAttachedToWindow and onDetachedFromWindow methods must call through to the like named methods of the returned LayeredDialogSessions (onAttachedToWindow, onDetachedFromWindow).

  • The view's onSaveInstanceState and onRestoreInstanceState methods must call through to the like named methods of the returned LayeredDialogSessions (onSaveInstanceState, onRestoreInstanceState).
