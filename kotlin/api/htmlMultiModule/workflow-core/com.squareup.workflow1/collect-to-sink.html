collectToSink

suspend fun <T, PropsT, StateT, OutputT> Flow<T>.collectToSink(actionSink: Sink<WorkflowAction<PropsT, StateT, OutputT>>, handler: (T) -> WorkflowAction<PropsT, StateT, OutputT>)(source)

Collects from a Flow by converting each item into a WorkflowAction and then sending them to the actionSink. This may be used as an alternative to a asWorker for certain Flows.

Unlike merely calling collect yourself and emitting an action directly to the sink, this operator propagates back pressure from the workflow runtime, so if there is a lot of contention on the workflow runtime the flow will be suspended while the action is queued and only resumed after the action has been applied.

Example:

class MyWorkflow(
  private val myFlow: Flow<…>
) {

  // Omitting types for brevity.
  override fun render() {
    context.runningSideEffect("collector") {
      myFlow.collectToSink(context.actionSink) { value ->
        action("collect") { setOutput(value) }
      }
    }
  }
}
